# Hashtag Strategy Guide

Complete guide to hashtag optimization for maximum reach and engagement across all platforms.

## Platform-Specific Guidelines

### LinkedIn Hashtags

**Recommended Count:** 3-5 hashtags
**Placement:** End of post
**Character Impact:** Minimal (3,000 char limit)

**Strategy:**
- Mix of broad + niche hashtags
- Industry-relevant terms
- Trending professional topics
- Location-based (if relevant)

**LinkedIn Hashtag Tiers:**

**Tier 1: High Competition (1M+ followers)**
- #Business
- #Marketing
- #Technology
- #Innovation
- #Leadership

**Tier 2: Medium Competition (100K-1M followers)**
- #SmallBusiness
- #Entrepreneurship
- #DigitalMarketing
- #SaaS
- #Startup

**Tier 3: Niche (10K-100K followers)**
- #B2BSales
- #MarketingAutomation
- #ProductManagement
- #CustomerSuccess
- #GrowthHacking

### Facebook Hashtags

**Recommended Count:** 1-3 hashtags
**Placement:** End of post or naturally in text
**Character Impact:** Minimal (63,206 char limit)

**Strategy:**
- Use sparingly compared to other platforms
- Brand-specific hashtags
- Campaign hashtags
- Local hashtags for businesses

**Facebook Hashtag Best Practices:**

## Hashtag Research Process

### 1. Identify Seed Hashtags

Start with 5-10 core hashtags related to your business:

**Example for SaaS Company:**
```
#SaaS #BusinessSoftware #Automation #Productivity #B2B
```

### 2. Expand Using Tools

**LinkedIn:**
- Type hashtag in search
- Check follower count
- Review related hashtags

**Instagram:**
- Search hashtag
- Check post count
- Explore "Related" section

**Twitter/X:**
- Check trending topics
- Use Twitter Analytics
- Monitor competitor hashtags

### 3. Analyze Competition

For each hashtag, check:
- **Volume:** How many posts?
- **Competition:** How active?
- **Relevance:** Matches your content?
- **Engagement:** Are posts performing well?

### 4. Create Hashtag Sets

Build reusable sets for different content types:

**Set 1: Product Posts**
```
LinkedIn: #ProductInnovation #SaaS #BusinessSolutions #Technology #B2B
Facebook: #NewProduct #YourBrand
Instagram: [30 hashtags mixing high/medium/low competition]
Twitter: #ProductLaunch #SaaS
```

**Set 2: Company Culture**
```
LinkedIn: #CompanyCulture #TeamWork #Innovation #Leadership #WorkLife
Facebook: #TeamLife #YourBrand
Instagram: [30 hashtags about team, culture, work life]
Twitter: #TeamWork #Culture
```

**Set 3: Thought Leadership**
```
LinkedIn: #ThoughtLeadership #BusinessStrategy #Innovation #FutureOfWork #Industry
Facebook: #BusinessInsights #YourBrand
Instagram: [30 hashtags about industry, insights, business]
Twitter: #ThoughtLeadership #Industry
```

## Banned and Spam Hashtags

### Instagram Banned Hashtags

Avoid these types (list changes frequently):
- Adult content related
- Drug/substance related
- Generic spam tags (#like4like, #follow4follow)
- Overly generic (#photography with millions of posts)

**Check Before Using:**
- Search hashtag on Instagram
- If no "Top Posts" section = likely banned
- Monitor for shadowban (posts not appearing in feed)

### Hashtag Spam Indicators

❌ **Avoid:**
- #like4like, #follow4follow, #tagsforlikes
- #instadaily, #instagood (overly generic)
- Hashtags unrelated to your content
- Using same 30 hashtags on every post

## Hashtag Testing Framework

### A/B Testing Hashtags

**Test Setup:**
1. Choose content type (e.g., product posts)
2. Create 2 hashtag sets
3. Post similar content with different sets
4. Compare results after 1 week

**Example Test:**

**Set A (Broad):**
```
#Business #Marketing #Technology #Innovation #Success
```

**Set B (Niche):**
```
#SaaS #B2BSales #MarketingAutomation #GrowthHacking #StartupLife
```

**Measure:**
- Impressions
- Engagement rate
- Profile visits
- Followers gained

**Decision:** Keep better performing set, retire other
