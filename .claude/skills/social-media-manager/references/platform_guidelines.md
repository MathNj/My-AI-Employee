# Platform Content Guidelines

Complete best practices for content creation, formatting, and engagement on each social media platform.

## Universal Principles

### Content Quality Standards

**All Platforms:**
- Clear, concise messaging
- Value-driven content
- Authentic voice
- Professional presentation
- Proofread before posting

**Visual Standards:**
- High-resolution images (minimum 1080px)
- On-brand colors and fonts
- Consistent visual style
- Proper aspect ratios
- Optimized file sizes

**Engagement Standards:**
- Respond within 2 hours
- Acknowledge all comments
- Professional tone always
- Never delete negative feedback (address it)
- Track sentiment and adjust

### LinkedIn Article Guidelines

**When to Post Articles vs Posts:**

**Post (Regular Update):**
- Quick insights
- Company updates
- Engagement questions
- Under 1,300 characters

**Article:**
- Deep-dive content
- How-to guides
- Thought leadership
- Over 1,300 characters

**Article Structure:**
1. Attention-grabbing title (40-60 characters)
2. Custom cover image (1200 x 627 px)
3. Strong opening paragraph
4. Subheadings every 2-3 paragraphs
5. Data and examples
6. Clear call-to-action

## Crisis Management Guidelines

### Negative Comments

**Response Framework:**

1. **Acknowledge** (within 2 hours)
```
"Thank you for bringing this to our attention. We're sorry you had this experience."
```

2. **Move to Private** (if complex)
```
"We'd like to understand more. Could you DM us your account details?"
```

3. **Resolve** (publicly if possible)
```
"Update: We've resolved this issue and implemented changes to prevent it."
```

4. **Never Delete** (unless spam/abuse)
- Deleting looks like hiding
- Address concerns transparently
- Show you care about feedback

### Major Issues

**Crisis Response Plan:**

1. **Pause Scheduled Posts**
- Review all queued content
- Remove anything insensitive
- Adjust timing

2. **Acknowledge Quickly**
```
"We're aware of [issue] and are investigating. We'll share updates as we
learn more."
```

3. **Be Transparent**
```
"Here's what happened, what we're doing about it, and how we'll prevent it
in the future."
```

4. **Follow Up**
```
"Update: [Resolution]. Thank you for your patience."
```

## Accessibility Guidelines

### Image Alt Text

**All Platforms:**
- Describe image content
- 125 characters or less
- Include relevant context
- Don't start with "Image of..."

**Example:**
```
Bad: "Image of people"
Good: "Marketing team celebrating Q4 results in conference room"
```

### Video Captions

**All Video Content:**
- Add closed captions
- Accurately transcribe speech
- Include sound descriptions
- Use high contrast text

### Color Contrast

**Text on Images:**
- Minimum 4.5:1 contrast ratio
- Use overlays for readability
- Test with colorblind simulators
- Avoid red/green combinations

## Legal and Compliance

### Required Disclosures

**Sponsored Content:**
```
Instagram: #ad #sponsored (top of caption)
Facebook: Use "Branded Content" tag
LinkedIn: Use "Promoted" disclosure
Twitter: #ad or #sponsored
```

**Affiliate Links:**
```
"This post contains affiliate links. We may earn commission from purchases."
```

### Copyright

**Images:**
- Own the rights
- Purchase stock images
- Use Creative Commons (with attribution)
- Never steal from Google Images

**Music:**
- Platform-licensed music only
- Own the rights
- Royalty-free music
- Avoid copyrighted songs

### Privacy

**User Content:**
- Get permission before reposting
- Credit original creator
- Don't share private information
- Follow GDPR/privacy laws
